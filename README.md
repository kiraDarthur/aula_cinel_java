# Java Class Activities - CINEL 2026

This repository contains all the practical exercises, school assignments, and Java projects developed during the training curriculum at **CINEL**.

## 📂 Repository Contents

The repository serves as a centralized archive for multiple independent Java projects:
*   **`projetojava3`**: A concert tour management system featuring custom graphic components, text file persistence, and date-sorting logic.
*   **`ficheiro3`**: An address book agenda tracking contacts, parsing file resources, and presenting clean layout windows.
*   **`cineelDoceSa` & `projeto3`**: Academic project baselines handling objective entities like clients and students.

## 🛠️ Built With

*   **Language**: Java SE 8+
*   **GUI Frameworks**: Java Swing & AWT (Both code-behind layouts and interactive form setups)
*   **Build Tool**: Apache Maven
*   **Storage Framework**: Plain Text File I/O Serialization (`.txt`)

## 💻 How to Run

1. Clone this entire project workspace to your local system:
   ```bash
   git clone https://github.com/kiraDarthur/aula_cinel_java.git
   ```
2. Open **NetBeans IDE** on your computer.
3. Select **File** ➔ **Open Project** and navigate into any specific subfolder workspace (e.g., `projetojava3`).
4. Right-click on your selected project node and select **Clean and Build**.
5. Click **Run** to launch the designated application.
